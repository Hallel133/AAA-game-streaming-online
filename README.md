# AAA Game Streaming Online

A guide on how to use various tools to play a game purchased from any platform on almost any other platform.

## Tools You Need

```plaintext
- Tailscale (optional): https://tailscale.com/download
- Sunshine (on the streaming device): https://github.com/LizardByte/Sunshine
- Moonlight (on the input device): https://github.com/moonlight-stream
- Chiaki (for PlayStation users only): https://sr.ht/~thestr4ng3r/chiaki/
```

All other guides are in another branch. Check them out.

I hope this guide helps you!
